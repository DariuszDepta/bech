# bech32 & bech32m encoder/decoder

## Setup

### Install Erlang

```shell
$ sudo dnf install erlang
```

of follow these [instructions](https://www.erlang.org/downloads).

### Install rebar3

Follow these [instructions](https://rebar3.org/docs/getting-started/).

### Create this library

```shell
$ rebar3 new lib bech
```
